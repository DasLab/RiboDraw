Welcome to RiboDraw's documentation for MATLAB scripts

Semiautomated layout of RNA tertiary structure diagrams
Version 0.6 (2017)

 

ABOUT THIS SOFTWARE

RiboDraw enables the quick drawing of diagrams representing helices and noncanonical pairs of complex RNA tertiary structures. All functions have been implemented in MATLAB.

The toolbox can be downloaded at the RiboKit Github page. There is a 'getting started' doc there too. Please send an e-mail if you have

  • questions or comments about the toolbox;
  • found a bug in the code;
  • developed your own algorithm, that you wish to incorporate into the toolbox

ABOUT THE DOCUMENTATION

The bulk of this documentation was produced automatically from the Matlab m-files of the toolbox using m2html and this nice template.
