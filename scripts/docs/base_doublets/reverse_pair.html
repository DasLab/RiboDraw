DESCRIPTION 

  base_pair = reverse_pair( base_pair );

  Switch the nucleotide partners involved in the base pair.

  Input:
   base_pair = struct with resnum1, chain1, resnum2, chain2

  Output:
   new_pair = same as input, but switching residues 1 and 2.
