PURPOSE

DESCRIPTION 

  base_stack = reverse_stack( base_stack );

  Switch the nucleotide partners involved in the base stack.

  Input:
   base_stack = struct with resnum1, chain1, resnum2, chain2

  Output:
   new_stack = same as input, but switching residues 1 and 2.
