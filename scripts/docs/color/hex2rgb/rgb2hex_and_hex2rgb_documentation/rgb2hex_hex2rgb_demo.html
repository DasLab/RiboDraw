0002 % These functions convert RGB color triplets to hexadecimal format, or
0003 % hexadecimal format to RGB color triplets.

0005 %% Syntax
0006 %
0007 %  rgb2hex(rgb)
0008 %  hex2rgb(hex)
0009 %  hex2rgb(hex,range)

0011 %% Description
0012 %
0013 % |rgb2hex(rgb)| converts RGB color triplets to hexadecimal format. If no
0014 % value in the |rgb| matrix exceeds unity, values are assumed to be scaled
0015 % from 0 to 1.  If any value in |rgb| exceeds unity, values are assumed to
0016 % be scaled from 0 to 255.
0017 %
0018 % |hex2rgb(hex)| converts hexadecimal to rgb color triplets scale from 0 to
0019 % 1.
0020 %
0021 % |hex2rgb(hex,range)| specifies a range as 1 or 256 for scaling output RGB values.
0022 % A previous version of this function required a range of 255 when scaling from 0 to 255,
0023 % but letting range equal 256 may be more intuitive, so either input will work now.
0024 % Whether you enter a range of 255 or 256, RGB values will be scaled from 0 to 255.
0025 % Default range is 1 to Matlab syntax, meaning values are scaled from 0 to 1.
0026 
0027 %% Examples of |rgb2hex|
0028 % What is the hexadecimal value of pure green?
0029 
0030 greenHex = rgb2hex([0 1 0])
0031 
0032 %%
0033 % What if the rgb values are scaled from 0 to 255?
0034 
0035 greenHexIsStill = rgb2hex([0 255 0])
0036
