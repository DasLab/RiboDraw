DESCRIPTION 

 set_bg_color( color )

 set background color. 

 INPUT
  color = [optional] text color like 'white' or RGB value like [1,1,1]
           if not specified, check plot_settings.bg_color. if that is not
           specified use [1,1,1] (white).
